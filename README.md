# Alcohol Detector

##  Overview
A device that detects alcohol level by blowing on the sensor and sends the data to Web Server

##  Components
- Arduino Uno
- MQ-3 Sensor
- ESP8266
- Jumper Wires
- Buzzer

##  Features
- Detects Alcohol Level
- Sends Data to Web Server
- Buzz when Alcohol Exceeds the Set Level

##  Challenges I Solved
- Calibration of MQ3 Sensor
- Arduino and ESP Serial Connection
- Sensor misreading → adjusted threshold values
